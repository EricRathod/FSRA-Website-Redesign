# FSRA Website Redesign

This project is a modern front-end redesign concept for the Financial Services Regulatory Authority of Ontario website.

## Features

- Responsive navigation bar
- Mobile-friendly layout
- Dark mode toggle
- Risk assessment tool
- FAQ accordion
- Contact form validation
- Clean government-style UI
- Accessibility-focused structure

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Purpose

This project was created as a portfolio project for front-end web design and financial services regulatory technology roles. It focuses on accessibility, usability, responsive design, and simple risk-based decision logic.

## How to Run

1. Download or clone the project.
2. Open the folder in VS Code.
3. Install Live Server extension.
4. Right-click `index.html`.
5. Click `Open with Live Server`.
